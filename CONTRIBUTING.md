# Contributing

Thanks for your interest in contributing to **Awesome AI Coding Assistants**!
This is a community-driven map of references for configuring AI coding assistants.

---

## What can I add?

- Repos of skills, instructions, rules, prompts or agents for AI coding assistants
- SDD tools related to AI-assisted development
- Videos, posts or talks about the topic

**Not accepted:**
- AI frameworks or ML libraries (LangChain, LlamaIndex, etc.)
- ML models or model repos
- General AI tools not related to coding assistant configuration

---

## How to contribute

1. Fork this repo
2. Add your resource in the correct section of `README.md`
3. Follow the entry format below
4. Open a Pull Request with a short description of what you're adding

---

## Entry format

```markdown
- **[owner/repo](https://github.com/owner/repo)** `tag1` `tag2`  
  > One-line description of what it is and why it's relevant
```

### Available tags

**Assistants** — only add if officially supported:

| Tag | Assistant |
|-----|-----------|
| `claude` | Claude Code — Anthropic |
| `copilot` | GitHub Copilot — GitHub |
| `cursor` | Cursor |
| `gemini` | Gemini CLI — Google |
| `codex` | Codex CLI — OpenAI |
| `windsurf` | Windsurf — Codeium |
| `opencode` | OpenCode |

**Pricing:**

| Tag | Meaning |
|-----|---------|
| `free` | Has a free tier |
| `paid` | Paid only |

> Only add tags you are certain about. When in doubt, leave them out.

## Adding content (videos, posts, talks)

Add to the **Content → Community** section:

```markdown
- **[Title](https://link)** — Your Name / Channel
```
