# Contribuindo

> 🇺🇸 [English version](CONTRIBUTING.md)

Obrigado pelo interesse em contribuir com o **Awesome AI Coding Assistants**!
Este é um mapa colaborativo de referências para configurar AI coding assistants.

---

## O que posso adicionar?

- Repos de skills, instructions, rules, prompts ou agents para AI coding assistants
- Ferramentas de SDD relacionadas ao desenvolvimento assistido por IA
- Vídeos, posts ou talks sobre o tema

**Não será aceito:**
- Frameworks de IA ou bibliotecas de ML (LangChain, LlamaIndex, etc.)
- Repos de modelos de ML
- Ferramentas de IA em geral não relacionadas à configuração de coding assistants

---

## Como contribuir

1. Faça um fork deste repo
2. Adicione seu recurso na seção correta do `README.md` **e** do `README.pt-br.md`
3. Siga o formato de entrada abaixo
4. Abre um Pull Request com uma descrição curta do que está adicionando

---

## Formato de entrada

```markdown
- **[owner/repo](https://github.com/owner/repo)** `tag1` `tag2`  
  > Descrição curta em uma linha do que é e por que é relevante
```

## Adicionando conteúdo (vídeos, posts, talks)

Adicione na seção **Conteúdo → Comunidade**:

```markdown
- **[Título](https://link)** — Seu Nome / Canal
```
