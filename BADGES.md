## Usage Rule

**Always use image badges instead of plain text tags in README.md and README.pt-br.md.**

Example:
```markdown
- **[owner/repo](https://github.com/owner/repo)** ![claude](https://img.shields.io/badge/claude-FF6B35?style=flat) ![free](https://img.shields.io/badge/free-22C55E?style=flat)  
  > Description
```
